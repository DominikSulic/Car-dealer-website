Opis projektnog zadatka

U ovome projektnome zadatku cilj je bio napraviti stranicu koja će omogućiti korisnicima rezerviranje servisa u određenim terminima, na određenim lokacijama, u određenim autokućama. Dok bi registrirani korisnik mogao kreirati zahtjeve za servisom, administrator i moderator imaju malo proširene opcije. Moderator može kreirati termine i mijenjati korisničke zahtjeve, dok admin može kreirati autokuće, lokacije, moderatore....

Bitne odrednice projektnog rješenja (ERA)




Popis i opis skripti, mapa mjesta, navigacijski dijagram

U projektu se nalaze direktoriji css, multimedija, javascript i php. Svaki direktorij, osim multimedije, sadrži datoteke s pripadajućim ekstenzijama.



Popis i opis korištenih tehnologija i alata

U ovome projektu su korišteni jquery, ajax, javascript općenito, php, html, css, mysql te naravno server koji nam je dodjeljen. Ajax se vecinom koristio



Popis i opis vanjskih (tuđih) modula/biblioteka i njihovo korištenje u skriptama i sl.

Jedino što je korišteno je od Google-a api za korištenje kod registracije u svrhu negiranja spam botova.
